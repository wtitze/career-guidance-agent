# Career Guidance Agent AI

Agente AI per l'orientamento post-diploma degli studenti.

## Struttura
- `frontend/`: Applicazione Next.js
- `backend/`: API FastAPI con agente AI
- `data/`: Database e cache

## Avvio rapido
1. `cd backend && pip install -r requirements.txt`
2. `cd frontend && npm install`
3. Avvia backend: `uvicorn app.main:app --reload`
4. Avvia frontend: `npm run dev`
